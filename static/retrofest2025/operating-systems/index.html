UNIX Operating Systems at RetroFest 2025

Here are the UNIX operating systems you can (hopefully) see in our little UNIX corner of RetroFest 2025. Apologies if any of these are missing - the machines are very old and fragile and I fear some might not survive the weekend.

Silicon Graphics IRIX 6.2

Running on the POWER Indigo 2.

IRIX 6.2 is a UNIX for the 64-bit MIPS IV architecture, based on UNIX System V Release 4. It uses ELF executables, like Linux does. It was released in 1996, and replaced by IRIX 6.5 in 1998 (IRIX 6.5 was the last version and was supported until 2013). It comes with the Indigo Magic Desktop.

You can read the IRIX 6.2 Technical Report here.

Sun Solaris 7

Running on the Sun Ultra 80.

Solaris 7 is a UNIX for 32-bit and 64-bit SPARC, as well as x86, based on UNIX System V Release 4. It uses ELF executables, like Linux does. It was released in November 1998 and supported until August 2008. It comes with OpenWindows and the Common Desktop Environment.

HP-UX 10.20

Running on the HP 9000 Visualize B132L+.

HP-UX 10.20 is a UNIX for 32-bit PA-RISC, released in 1996. It includes a Logical Volume Manager (LVM) and a journalled filesystem (VxFS). It is based on UNIX System V Release 4. It was replaced by HP-UX 11.00 but was supported until 2003. It comes with HP VUE desktop and the Common Desktop Environment.

HP-UX 11.00

Running on the HP 9000 Visualize C3000.

HP-UX 11.00 is a UNIX for 32-bit and 64-bit PA-RISC, released in 1997. It is based on UNIX System V Release 4. Compared to HP-UX 10.20, it adds full 64-bit addressing, and drops the HP VUE desktop, leaving CDE as the only option. It was replaced by HP-UX 11.11 (also known as HP-UX 11i) but was supported until 2006. It comes with the Common Desktop Environment.

RISC iX 1.2.1

Running on the Acorn R260.

RISC iX was first releaseed in 1988. It is a port of BSD 4.3 to the ARM architecture with extra features such as System V extensions, the X Window System (X11R2 later R4), the Norcroft C Compiler, and Sun's NFS. It provided a reasonably complete UNIX system for the early 1990's.

RISC iX 1.2 replaces 1.1's IXI's X.desktop with the Motif Window Manager as the default, though IXI's desktop tools remain available.

Digital UNIX V4.0

Available on the DEC 3000 Model 800 AXP.

Digital UNIX, formerly DEC OSF/1 AXP and subsequently renamed again to Tru64 UNIX, is a 64-bit operating system for the Alpha architecture. It was first released in March 1993. The OS is based on the Mach modular kernel, hosting Unix services in user space. It comes with the DECwindows Motif desktop and the Common Desktop Environment.

Non-UNIX Operating Systems at RetroFest 2025

OpenVMS V7.2

Running on the DEC 3000 Model 800 AXP.

OpenVMS, often referred to as just VMS, as it was called originally, was developed by Digital Equipment Corporation for their line of VAX minicomputers. It was first released in August 1978. It comes with the DECwindows Motif desktop and the Common Desktop Environment.

VMS is fairly well known for its clustering and high availability. It is still supported to this day for a small number of serious users who can't quite migrate just yet.

After the VAX, VMS was ported to DEC Alpha and Intel Itanium, both of which died shortly thereafter. What that spells for the x86-64, which VMS has just recently been ported to, remains to be seen.
