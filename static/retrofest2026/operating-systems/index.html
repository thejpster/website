UNIX Operating Systems at RetroFest 2026

Here are the UNIX operating systems you can (hopefully) see in our little UNIX corner of RetroFest 2026. Apologies if any of these are missing - the machines are very old and fragile and I fear some might not survive the weekend.

Silicon Graphics IRIX 6.5

IRIX 6.5 is SGI's UNIX for their 32-bit and 64-bit MIPS machines, based on UNIX System V Release 4. It uses ELF executables, like Linux does. It was released in 1998, and was the last major version IRIX to be released, with technical support available until the end of 2013. It came with the Indigo Magic Desktop.

Sun Solaris 7

Solaris 7 is a UNIX for 32-bit and 64-bit SPARC, as well as x86, based on UNIX System V Release 4. It uses ELF executables, like Linux does. It was released in November 1998 and supported until August 2008. It came with OpenWindows and the Common Desktop Environment.

Sun Solaris 9

Solaris 9 is a UNIX for 32-bit and 64-bit SPARC, as well as x86, based on UNIX System V Release 4. It is fully backwards-compatible with older releases all the way back to Solaris 2.5. It was released in May 2002 and supported until October 2014. It came with GNOME 2 and the Common Desktop Environment.

HP-UX 10.20

HP-UX 10.20 is a UNIX for 32-bit PA-RISC, released in 1996. It includes a Logical Volume Manager (LVM) and a journaled filesystem (VxFS). It is based on UNIX System V Release 4. It was replaced by HP-UX 11.00 but was supported until 2003. It came with HP VUE desktop and the Common Desktop Environment.

NeXTSTEP 3.3

When Steve Jobs left Apple he start two companies - NeXT and Pixar. NeXT made custom Motorola 68K workstations running NeXTSTEP OS - an Objective-C based mash-up of BSD UNIX and the Mach microkernel from Carnegie Mellon University. When they finally realised that it was not profitable to make make wildly expensive bespoke high-end workstations, they pivoted to being a software company and NeXTSTEP was ported to x86, PowerPC, SPARC and PA-RISC.

In 1994 NeXT started working with Sun to produce OpenStep, which was the NeXTSTEP SDK ported to run on other operating system kernels, like Windows NT. This allowed some level of application portability. Confusingly the OpenStep UI and SDK was available on a NeXTSTEP based OS called OPENSTEP - which was basically NeXTSTEP 4.

NeXTSTEP includes a unique desktop environment which, via OPENSTEP, Rhapsody, Mac OS X Server 1.0 and Mac OS X 10.x, you still see echoes of it in today's macOS.

Apple Mac OS X Server 1.0 (Codename: Rhapsody)

Steve Jobs returned to Apple when they bought his company, NeXT. With it, Apple acquired the OPENSTEP operating system and SDKs. Apple had been busy failing to replace the classic Macintosh operating system and this gave them a way out. By merging an OPENSTEP base with classic Mac OS compatibility and UI concepts, they formed Mac OS X - an operating system that is still with us over 25 years later.

Our machine is running Mac OS X Server 1.0, a hurriedly finished commercial release of the Rhapsody prototype OS, which acts as stepping stone between OPENSTEP and Mac OS X 10.0. It was sold as an 'enterprise' product because it had zero backwards compatibility with classic Mac OS applications, and Apple belatedly realised that that was a bad thing to foist upon unsuspecting users. Mac OS X Server 1.0 was released in March 1999, while the full Mac OS X 10.0 desktop release (with support for the classic Mac OS programs you already had) didn't arrive until June 2001.

Mac OS X Server has a desktop that is somewhere between Mac OS 8 and OPENSTEP.
