UNIX Operating Systems at RetroFest 2026

Here are the UNIX operating systems you can (hopefully) see in our little UNIX corner of RetroFest 2026. Apologies if any of these are missing - the machines are very old and fragile and I fear some might not survive the weekend.

Silicon Graphics IRIX 6.5

IRIX 6.5 is SGI's UNIX for their 32-bit and 64-bit MIPS machines, based on UNIX System V Release 4. It uses ELF executables, like Linux does. It was released in 1998, and was the last major version IRIX to be released, with technical support available until the end of 2013. It came with the Indigo Magic Desktop.

Sun Solaris 7

Solaris 7 is a UNIX for 32-bit and 64-bit SPARC, as well as x86, based on UNIX System V Release 4. It uses ELF executables, like Linux does. It was released in November 1998 and supported until August 2008. It came with OpenWindows and the Common Desktop Environment.

Sun Solaris 9

Solaris 9 is a UNIX for 32-bit and 64-bit SPARC, as well as x86, based on UNIX System V Release 4. It is fully backwards-compatible with older releases all the way back to Solaris 2.5. It was released in May 2002 and supported until October 2014. It came with GNOME 2 and the Common Desktop Environment.

HP-UX 10.20

HP-UX 10.20 is a UNIX for 32-bit PA-RISC, released in 1996. It includes a Logical Volume Manager (LVM) and a journaled filesystem (VxFS). It is based on UNIX System V Release 4. It was replaced by HP-UX 11.00 but was supported until 2003. It came with HP VUE desktop and the Common Desktop Environment.

HP-UX 11.00

HP-UX 11.00 is a UNIX for 32-bit and 64-bit PA-RISC, released in 1997. It is based on UNIX System V Release 4. Compared to HP-UX 10.20, it adds full 64-bit addressing, and drops the HP VUE desktop, leaving CDE as the only option. It was replaced by HP-UX 11.11 (also known as HP-UX 11i) but was supported until 2006. It came with the Common Desktop Environment.

HP-UX 11.11 (aka HP-UX 11i v1)

HP-UX 11.11 (or 11i v1) was the last HP-UX for desktop PA-RISC systems (there was an 11i v2 for servers, but it has no GUI). It came with the Common Desktop Environment and basically looks exactly like 11.00.

Apple Mac OS X Server 1.0 (Codename: Rhapsody)

Steve Jobs returned to Apple when they bought his company, NeXT. With it, Apple acquired NeXTSTEP - an Objective-C based mash-up of BSD UNIX and the Mach microkernel from Carnegie Mellon University. That OS started on Motorola 68K but had also been ported to x86, PowerPC, SPARC and PA-RISC. Apple had previously tried and failed to replace the classic Macintosh operating system - Mac OS. But by merging a NeXTSTEP base with classic Mac OS compatibility and UI concepts, they formed Mac OS X - an operating system that is still with us over 25 years later.

Our machine is running Mac OS X Server 1.0, a stepping stone between NeXTSTEP and Mac OS X 10.0. It was sold as an 'enterprise' product because it had zero backwards compatibility with classic Mac OS applications, and Apple belatedly realised that that was a bad thing to foist upon unsuspecting users. Mac OS X Server 1.0 was released in March 1999, while the full Mac OS X 10.0 desktop release didn't arrive until June 2001.

As you click around see if you can see which parts come from Mac OS 9, which parts come from NeXTSTEP, and which parts made it into the Mac OS X we came to know and love. There are even UI elements here you'd recognise in macOS 26 (Tahoe), a modern RISC UNIX I use every day.
